Maze and Cave Generator
Проект "Maze" - это инструмент для генерации, отрисовки и решения лабиринтов и пещер, разработанный на C++ с использованием Qt для графического интерфейса и Google Test для тестирования.

Функциональность
Генерация лабиринтов: Создание идеальных лабиринтов с одним уникальным путем между любыми двумя точками.
Генерация пещер: Применение клеточных автоматов для создания ветвящихся структур пещер.
Отрисовка и визуализация: Графический интерфейс для просмотра и анализа сгенерированных структур.
Поиск решения: Алгоритмы для нахождения кратчайших путей в лабиринтах.

Основные Цели Makefile
all: Компилирует и запускает весь проект.
install: Собирает проект с использованием qmake и запускает его.
open: Открывает папку с программой.
uninstall: Удаляет собранный проект.
dist: Создает архив с исходным кодом проекта.
test: Запускает модульные тесты.
gcov: Генерирует отчет о покрытии кода тестами.
clean: Удаляет временные файлы, созданные во время сборки или тестирования.
leak: Проверяет программу на утечки памяти.
dvi: Открывает документацию проекта. Это ваша цель для доступа к документации.

Инструкции по Установке
Клонируйте репозиторий.
В корне проекта выполните make install для сборки.
Запустите приложение командой make open.
